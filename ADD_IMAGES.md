# How to Add Product Images

## Image Directory Structure

Place your product images in the following directory:
```
client/public/images/products/furor-sport/
```

## Required Images

Add the following image files with these exact names:

1. **furor-black-compression-short.jpg** (or .png)
   - For: FUROR SPORT Black Compression Shirt - Short Sleeve
   - Black short-sleeved compression shirt

2. **furor-grey-compression-long.jpg** (or .png)
   - For: FUROR SPORT Grey Compression Shirt - Long Sleeve
   - Grey long-sleeve compression shirt

3. **furor-wolf-design.jpg** (or .png)
   - For: FUROR SPORT Wolf Design Athletic Shirt - Long Sleeve
   - Long-sleeved shirt with wolf graphic

4. **furor-dragon-design.jpg** (or .png)
   - For: FUROR SPORT Dragon Design Athletic Shirt - Long Sleeve
   - Long-sleeved shirt with purple dragon design

5. **furor-geometric-tshirt.jpg** (or .png)
   - For: FUROR SPORT Geometric Pattern T-Shirt - Black
   - Black t-shirt with geometric patterns

## After Adding Images

Once you've added the images to the directory, the products will automatically use them. The images will be accessible at:
- `/images/products/furor-sport/furor-black-compression-short.jpg`
- `/images/products/furor-sport/furor-grey-compression-long.jpg`
- etc.

## Image Recommendations

- Format: JPG or PNG
- Size: 500x500px to 1000x1000px recommended
- File size: Keep under 500KB for faster loading
