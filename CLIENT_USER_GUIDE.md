## üì¶ How to Add Products

### Step-by-Step Guide:

1. **Login to Admin Portal**
   - Go to admin login page
   - Enter your password

2. **Go to Products Section**
   - Click "Products" in the left menu
   - Or click "Manage Products" button on dashboard

3. **Click "Add New Product" Button**
   - You'll see a form to fill in

4. **Fill in Product Information:**

   **Basic Information:**
   - **Product Name:** Enter the name (e.g., "Fitness Compression Shorts")
   - **Description:** Write a detailed description
   - **Category:** Select from dropdown (e.g., "Fitness Wears")
   - **Subcategory:** (Optional) Select if available
   - **Price:** Enter the regular price (e.g., 29.99)
   - **Sale Price:** (Optional) Enter if product is on sale (e.g., 24.99)
   - **Stock:** Enter how many items you have (e.g., 100)

   **Product Image:**
   - Choose "Upload from Device" or "Enter Image URL"
   - If uploading: Click "Choose File" and select image
   - If using URL: Paste the image link
   - **Add Multiple Images:** Click "Add Another Image" to upload more photos
   - Customers can scroll through all images on product page

   **Product Variants (Size & Color):**
   - **Add Sizes:** Click "Add Size" and enter sizes (e.g., S, M, L, XL)
   - **Add Colors:** Click "Add Color" and enter colors (e.g., Black, Blue, Red)
   - Customers can select size and color when viewing the product

5. **Featured Product (Star Feature):**
   - **Check the "Featured" checkbox** ‚≠ê
   - When checked, this product will appear on the homepage
   - Featured products show in the hero section with:
     - Large product image
     - Product title and description
     - Key features (shown as checkmarks ‚úì)
     - Price information
     - "Shop Now" button

6. **Click "Add Product"**
   - Product will be saved
   - It will appear in your products list
   - If featured, it will show on homepage automatically

## üìã Managing Orders

### Viewing Orders:

1. **Go to Orders Section**
   - Click "Orders" or "Order Management" in admin menu
   - You'll see a list of all orders

2. **Order Information Displayed:**
   - Order Number
   - Customer Name
   - Order Date
   - Total Amount
   - Order Status (Pending, Processing, Shipped, Delivered, Cancelled)
   - Payment Status

3. **View Order Details:**
   - Click "View" button next to any order
   - See complete order information:
     - Customer details (name, email, phone, address)
     - All products ordered (with images, sizes, colors, quantities)
     - Total amount
     - Order status

4. **Update Order Status:**
   - Click "View" on an order
   - Change status using dropdown:
     - **Pending:** Order just received
     - **Processing:** Order is being prepared
     - **Shipped:** Order has been shipped
     - **Delivered:** Order delivered to customer
     - **Cancelled:** Order cancelled
   - Click "Update Status"

5. **Export Shipping Label:**
   - Click "View" on an order
   - Click "Export Label" button
   - A print-friendly shipping label opens
   - Contains:
     - Customer address
     - Order number
     - Product list with images
     - All order details
   - Print it and attach to package

6. **Delete Orders:**
   - You can delete delivered or cancelled orders
   - Click the delete button (üóëÔ∏è) next to the order
   - Confirm deletion
   - This helps keep your order list clean

7. **Export to CSV:**
   - Click "Export CSV" button at the top
   - Downloads all orders as Excel/CSV file
   - Useful for record keeping or analysis

## üé® Important Features Explained

### 1. Product Images

**Multiple Images:**
- You can add multiple images per product
- Customers see an image carousel (with arrows to navigate)
- First image is the main product image
- Additional images show different angles or variations

**How to Add:**
- Click "Add Another Image" button
- Upload or enter URL for each image
- Images appear in order you add them

### 2. Product Variants (Size & Color)

**What are Variants?**
- Different options for the same product
- Example: Same t-shirt in different sizes (S, M, L) and colors (Black, Blue)

**How It Works:**
- Add sizes: S, M, L, XL, etc.
- Add colors: Black, Blue, Red, etc.
- Customers select size and color when viewing product
- Selected size/color is saved with the order

### 3. Sale Prices

**How to Set Sale Price:**
- Enter regular price in "Price" field
- Enter lower price in "Sale Price" field
- Product shows:
   - Sale price in large, red text
   - Original price crossed out
   - "SALE" badge on product image

**Example:**
- Regular Price: $29.99
- Sale Price: $24.99
- Customer sees: ~~$29.99~~ **$24.99** with SALE badge

### 4. Stock Management

**Stock Levels:**
- Enter how many items you have in stock
- System tracks inventory
- Low stock alert: Products with less than 10 items show in "Low Stock" filter

**View Low Stock:**
- Go to Products section
- Click "Low Stock Items" filter
- See all products with stock < 10

### 5. Featured Products (Homepage Display)

**What Makes a Product Featured:**
- Check the "Featured" checkbox when adding/editing product
- Featured products appear in homepage hero section

**Hero Section Features:**
- Large product image
- Product title and category
- Description text
- Key features as checkmarks (automatically extracted):
  - ‚úì Feature 1
  - ‚úì Feature 2
  - ‚úì Feature 3
  - etc.
- Price information
- "Shop Now" button

**Multiple Featured Products:**
- You can feature multiple products
- They rotate automatically every 5 seconds
- Customers can click arrows to navigate
- Shows indicators (dots) at bottom

### 6. Search & Filters

**For Products:**
- Search by product name or description
- Filter by category
- Filter by low stock items

**For Orders:**
- Filter by order status (Pending, Processing, etc.)
- View all orders or specific status

**For Customers:**
- Search by name, email, phone, city, or country

### 7. WhatsApp Ordering

**How It Works:**
1. Customer adds products to cart
2. Goes to checkout
3. Fills in information
4. Clicks "Place Order via WhatsApp"
5. WhatsApp opens with formatted message containing:
   - Order number
   - Customer details
   - All products with images, sizes, colors, quantities
   - Total amount
6. Customer sends message to complete order

**Order Notification:**
- You receive notification in admin dashboard
- Order appears in Orders section with "Pending" status
- You can update status as you process the order
